Database client:
Free multi-platform database tool for developers, database 
administrators, analysts and all people who need to work with databases. 
Supports all popular databases: MySQL, PostgreSQL, SQLite, Oracle, DB2, 
SQL Server, Sybase, MS Access, Teradata, Firebird, Apache Hive, Phoenix, 
Presto, etc.
